# ==================== 核心概念 ====================

concepts:
  - name: 语义虚拟机 (Semantic VM)
    description: |
      LLM 作为处理器，自然语言作为"机器码"，PEF 作为可执行文件。
      语义 VM 是 IntentOS 的核心，它将自然语言意图编译为 LLM 可执行的 Prompt。
    keywords: [LLM, Prompt, PEF, 语义编译]
    
  - name: Self-Bootstrap
    description: |
      系统可以修改自身的指令集、编译规则和执行逻辑。
      通过元意图（Meta-Intent）驱动系统自我演进。
    keywords: [自我演进，元意图，自修改]
    
  - name: 分布式内核
    description: |
      多节点集群中，每个节点运行独立的语义 VM，
      通过 HTTP RPC 通信，跨网络组成整体 OS。
    keywords: [分布式，集群，RPC, Map-Reduce]
    
  - name: 意图编译器
    description: |
      将自然语言编译为 LLM Prompt，支持三级缓存和优化器。
      7 层套娃分层处理：意图解析 → 任务规划 → 上下文收集 → 安全验证 → 能力绑定 → 执行 → 改进
    keywords: [编译器，缓存，优化，7 层处理]
    
  - name: PEF (Prompt Executable File)
    description: |
      语义 VM 的"机器码"，包含系统 Prompt、意图指令和上下文。
      可跨节点传输和执行。
    keywords: [可执行文件，语义指令]
    
  - name: 7 层套娃分层编译器
    description: |
      L1: 意图解析 → L2: 任务规划 → L3: 上下文收集 → 
      L4: 安全验证 → L5: 能力绑定 → L6: 执行 → L7: 改进
    keywords: [分层，编译器，处理流程]

# ==================== 快速开始 ====================

quick_start:
  # 安装
  install: |
    pip install pyyaml
    
  # 启动 Shell
  run_shell: |
    PYTHONPATH=. python intentos/interface/shell.py
    
  # 启动 API 服务器
  run_api: |
    PYTHONPATH=. python intentos/interface/api.py
    
  # Python 代码使用
  code_example: |
    from intentos import SemanticVM, create_executor
    
    # 创建执行器
    executor = create_executor(provider="openai", api_key="sk-...")
    
    # 创建语义 VM
    vm = SemanticVM(executor)
    
    # 执行程序
    result = await vm.execute_program("my_program")
