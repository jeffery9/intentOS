# IntentOS - 工业级 AI 原生操作系统 (v2.1 融合版)

> **语言即系统 · 语义冯·诺依曼架构 · 分布式自举内核**

IntentOS 是一个**从第一性原理出发**构建的 AI 原生操作系统。它将 LLM 抽象为语义处理器（CPU），将自然语言意图编译为可执行的机器指令（Opcode），并提供完整的分布式、安全隔离及资源配额机制。

---

## 🧠 核心理念：道即 Meta

| 东方哲学 | 西方 CS 体系 | IntentOS 对应 |
|----------|-------------|---------------|
| **器** (万物) | Instance/Data | 具体的业务数据、运行中的进程 |
| **法** (规则) | Meta/Metadata | 意图定义、Prompt 模板、Schema |
| **道** (本源) | **Meta-Meta** | 语义 VM、Self-Bootstrap、演进算法 |

> 💡 **道生一，一生二，二生三，三生万物**  
> 在 IntentOS 中：**语义 VM → 意图/能力 → 进程/节点 → 分布式应用生态**

## 🚀 快速开始

### 安装
```bash
pip install -e .
```

### 启动边缘服务节点
任何节点都可以作为对外服务的入口：
```bash
# 在 8000 端口启动节点
PYTHONPATH=. python intentos/runtime/agent.py 8000
```

### 执行分布式意图 (v1 API)
```bash
curl -X POST http://localhost:8000/v1/execute 
     -H "Content-Type: application/json" 
     -d '{"intent": "并行处理大规模日志文件，提取所有 404 错误并汇总报告"}'
```
