## 传播机制

### 1. 病毒式传播 (Viral Transmission)

**核心**: 让每个用户成为传播者

```python
# 启动病毒式传播活动
python -m intentos.bootstrap.social_transmission --action viral
```

**传播要素**:
```
传播力 = 触达人数 × 转化率 × 病毒系数

病毒系数 (K) = 每个用户邀请的新用户数
K > 1: 病毒式增长
K < 1: 逐渐消失
```

**实施策略**:

#### A. 识别种子用户
```python
种子用户特征:
- 高影响力 (GitHub stars > 1000)
- 高活跃度 (每周 commit > 10)
- 高连接性 (follower > 500)
- 早期采用者倾向
```

### 3. 价值网络 (Value Network)

**核心**: 参与者越多，每个参与者价值越大

```python
# 创建价值网络
python -m intentos.bootstrap.social_transmission --action network
```

**梅特卡夫定律**:
```
网络价值 = 用户数²

1 个用户：价值 1
10 个用户：价值 100
100 个用户：价值 10,000
1000 个用户：价值 1,000,000
```

**网络参与者**:
```
        ┌──────────┐
        │ 投资者   │
        └────┬─────┘
             │ 资金
             ↓
┌──────────┐    ┌──────────┐
│ 开发者   │←→│ 用户     │
└────┬─────┘    └────┬─────┘
     │ 应用          │ 反馈
     ↓               ↓
        ┌──────────┐
        │ 生态     │
        └──────────┘
```

**价值流动**:
| 流向 | 价值 | 回报 |
|------|------|------|
| 开发者 → 用户 | 应用/插件 | 收入/反馈 |
| 用户 → 开发者 | 收入/反馈 | 应用/插件 |
| 投资者 → 生态 | 资金 | 增值回报 |
| 生态 → 投资者 | 增值 | 资金 |

**网络效应类型**:
1. **直接网络效应**: 用户越多越有价值 (如社交网络)
2. **间接网络效应**: 一边用户吸引另一边 (如平台)
3. **数据网络效应**: 数据越多产品越好 (如 AI)
4. **技术网络效应**: 集成越多越难替换 (如生态)
