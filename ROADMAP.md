### v8.5 - 分布式通信与接口层 (2026-03-13) ✅

- **分布式处理器**: 支持 REPLICATE, SPAWN, BROADCAST。
- **内核 Shell**: 实现 /ps, /top, /df, /mem, /nodes, /history。
- **REST API Gateway**: 提供 v1 版本执行、状态和存储接口。

### v9.0 - 分布式进程管理 (2026-03-13) ✅

- **分布式进程管理器**: 实现 PCB (进程控制块) 和全局进程表。
- **Fork/Exec 机制**: 实现语义程序的分布式派生、调度与实时追踪。
- **进程控制**: 实现 /kill 系统调用与进程自动回收。

### v12.0 - AI Native PaaS 与商业化 (2026-05-15) ✅

**新增核心模块**:

- **用量计量模块** (`intentos/agent/metering.py`):
  - 记录 CPU 时间、Token 消耗、执行时长、内存使用
  - 支持按量计费、订阅制、阶梯定价
  - 用量统计和账单生成

- **数字钱包模块** (`intentos/agent/wallet.py`):
  - 支持加密货币 (ETH/USDT) 和法币
  - 集成 MetaMask/ImToken/Stripe/支付宝
  - 自动充值和扣费
  - 订阅管理

- **应用市场模块** (`intentos/agent/marketplace.py`):
  - App 发布、审核、上架
  - 用户评价和评分
  - 收益分成 (开发者 80%、平台 15%、推荐者 5%)
  - 用量统计和收益报表

- **App 提交机制** (`intentos/agent/submission.py`):
  - Mainframe 模式：本地开发，云端执行
  - 意图包构建和验证
  - 能力注册和提交
  - 远程执行 API

**示范 App (5 个)**:

- 📝 **文档总结服务**: 按文档页数 $0.05/页
- 🌐 **多语言翻译**: 按字数 $0.01/100 字

**技术特性**:
- 云服务自动开通 (AWS/Azure/GCP/阿里云/腾讯云)
- 自动 Scale 系统 (基于负载/定时/预测性)
